# 3D Model Drop Zone

Place web-ready 3D assets here. Prefer `.glb` files because they bundle mesh, material, and textures in one file.

Expected paths:

```text
assets/models/stadiums/default_stadium.glb
assets/models/players/player_base.glb
assets/models/players/goalkeeper_base.glb
assets/models/balls/match_ball.glb
```

Optional texture folders:

```text
assets/textures/stadiums/
assets/textures/players/
assets/textures/balls/
```

Recommended limits:

- Stadium: 5-15 MB
- Player: 1-5 MB
- Ball: below 1 MB
- Textures: 1024px or 2048px `.png`, `.jpg`, or `.webp`
